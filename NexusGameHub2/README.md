🎮 Sobre o Projeto

O Arcade JavaScript é um portal interativo de jogos clássicos desenvolvido com HTML5, CSS3 e JavaScript, focado na prática de lógica de programação, manipulação do DOM e criação de interfaces dinâmicas sem recarregar a página.

O projeto reúne três jogos populares em uma única plataforma:

✊✋✌️ Jokenpô (Pedra, Papel ou Tesoura)
🎲 Batalha de Dados
🪙 Cara ou Coroa

Além da diversão, o objetivo principal foi aplicar conceitos fundamentais do desenvolvimento Front-End moderno, utilizando funções, eventos, estruturas condicionais, repetição e atualização dinâmica da interface.

🎮 Plataforma de Mini Jogos

Bem-vindo à Plataforma de Mini Jogos, um projeto desenvolvido com HTML, CSS e JavaScript que reúne jogos clássicos e divertidos em uma experiência moderna, dinâmica e totalmente interativa. Aqui, o jogador pode desafiar a máquina em partidas rápidas de Jokenpô, Cara ou Coroa e Batalha de Dados.

O objetivo do projeto é oferecer entretenimento simples, acessível e divertido para todas as idades, utilizando animações, efeitos visuais e lógica de programação para criar uma experiência fluida e envolvente diretamente no navegador.

🚀 Tecnologias Utilizadas
HTML5 → Estruturação da interface
CSS3 → Estilização, animações e efeitos visuais
JavaScript → Lógica dos jogos e interatividade
LocalStorage → Salvamento de dados e estatísticas
Canvas API → Efeitos visuais animados
🕹️ Jogos Disponíveis
✊ Jokenpô

Um clássico jogo de Pedra, Papel e Tesoura contra a CPU.

Destaques:
Sistema organizado utilizando objetos (CHOICES)
Feedback visual para vitória, derrota e empate
Interface dinâmica e intuitiva
Atualização automática do placar
🪙 Cara ou Coroa

Escolha entre cara ou coroa e teste sua sorte.

Destaques:
Animação da moeda
Efeitos visuais e banners de resultado
Sistema de sequência de vitórias (streak)
Bloqueio de múltiplos cliques durante animações
Salvamento de estatísticas com LocalStorage
🎲 Batalha de Dados

Um jogo com visual arcade/cyberpunk onde jogador e CPU disputam lançamentos de dados.

Destaques:
Interface neon inspirada em jogos retrô
Fundo animado estilo “Matrix” com Canvas API
Histórico de partidas
Placar automático
Efeitos glow, scanlines e animações futuristas
